# Power BI Sales Analysis

Projeto desenvolvido durante a formação Power BI Analyst da DIO.

## Objetivo

Criar um relatório de vendas utilizando Power BI, replicando páginas do curso e desenvolvendo uma terceira página com novos visuais.

## Visuais Desenvolvidos

* Mapa de vendas e unidades vendidas por país
* Mapa de lucro por país
* Gráfico de pizza/rosca com lucro por segmento

## Ferramentas Utilizadas

* Power BI Desktop
* GitHub
